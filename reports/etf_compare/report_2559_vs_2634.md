# 2559 vs 2634 ETF統計比較レポート

**分析日**: 2026年2月17日
**分析者**: 統計的手法による定量分析

---

## 結論サマリ

1. **共通期間（2021/3〜2026/2、約4.7年）の結論**: 2559（ACWI非ヘッジ）が2634（S&P500円ヘッジ）を**CAGR +11.8%pt、Sharpe +0.59、MDD 3.8%pt浅い**と全指標で圧倒。2559は全5暦年で勝利。
2. **統計的優位の有無**: 日次リターン差のt検定 p=0.049、Wilcoxon p=0.002、Jobson-Korkie-Memmel（Sharpe差）p=0.035、ブートストラップ2559優位確率98.8%。**5%水準で有意**。
3. **ただし「同一リスクの比較」ではない**: 2559はACWI+為替リスク、2634はS&P500+ヘッジコスト。実質的にこの差は「2021〜2025年の歴史的円安がもたらしたもの」であり、**円高局面では2634が逆転する**（レジーム分析で確認済み）。
4. **局面依存**: 円安局面ではSharpe差+4.8（2559圧勝）、円高局面ではSharpe差-4.0（2634逆転）。為替の方向が結論を支配する。

## B) 統計検定結果

### B-1) 平均リターン差の検定

| 頻度 | 平均差 | t統計量 | p値 | 判定 |

| 月次 | +0.775%/月 | 1.988 | 0.052 | 境界的 |

| ブートストラップ95%CI | — | [+0.091, +1.190] | **0を含まない** |

### B-3) ブートストラップ（ブロック、n=10,000、ブロック=21日）

| 指標 | 平均差 | 95%CI | 2559優位確率 |
|------|-------:|------:|------------:|

### B-4) 経済的有意性

- 年率CAGR差+11.8%ptは**経済的にも極めて大きい**
- ただしこの差の大部分は「為替（円安）」に起因。2021→2025年のUSD/JPYは約110→155円と40%超の円安が進行しており、為替ヘッジなしの2559に巨大な追い風だった
- **将来の為替が逆方向なら結論は逆転しうる**

## 最終判定

### 総合評価

| 基準 | 判定 |
|------|------|
| CAGR | **2559** (+11.8%pt) |
| MDD | **2559** (3.8%pt浅い) |
| Sharpe | **2559** (+0.59, p=0.035で有意) |
| Sortino | **2559** (+0.49) |
| Calmar | **2559** (+0.64) |
| 統計的有意性 | **あり**（ブートストラップ98.8%、JK検定p=0.035） |

### 条件別推奨

| 投資家タイプ | 推奨 | 理由 |
|:----------:|:---:|------|
| 長期積立（5年超） | **2559** | 為替は長期で平均回帰。分散が広く、ヘッジコスト不要 |
| 円高確信時の短期 | **2634** | 円高局面ではヘッジの恩恵でSharpe逆転 |
| 米国株集中投資したい | **2634** | ただし為替ヘッジなしのS&P500 ETF（例:1655）も検討すべき |
| リスク回避型 | **やや2559** | MDDが浅く、月次歪度も良好。ただし為替リスクは受容が前提 |

### 結論を変えうる条件

1. **円高トレンドへの転換**: USD/JPYが140円以下に向かう局面では2634が逆転する可能性が高い（レジーム分析で確認済み）
2. **日米金利差縮小**: ヘッジコスト低下は2634に追い風
3. **米国株一強の終焉**: ACWIの非米国部分がアンダーパフォームすれば2559は不利だが、現状では上昇・下落どちらでも2559が僅差で優位
4. **分析期間の特殊性**: 2021〜2025年は歴史的な円安・米国株上昇局面であり、この期間の結果を将来に外挿するのは危険

## 再現手順

### 依存ライブラリ
```
pip install yfinance pandas numpy scipy matplotlib
```

### 実行
```bash
python3 scripts/etf_compare_2559_2634.py
```

### データソース
- yfinance（Yahoo Finance Japan経由）
- 終値（Close）ベース。yfinanceのCloseは分配落ち後の値のため、別途分配金を取得してトータルリターン指数を自作

### 注意点
- **欠損処理**: 両ETFで取引日が一致しない場合はintersectionで共通日のみ使用
- **祝日**: 日本市場の祝日は自動的に除外（yfinanceが取引日のみ返す）
- **分配金反映**: 権利落ち日に分配金/前日終値を再投資リターンとして加算
- **為替ラグ**: 2559は海外指数を追跡するため、前日の海外市場の動きを翌日に反映するラグがある。相関分析では同日・ラグ付き両方を検証
- **ブートストラップ**: ブロックサイズ21日（約1ヶ月）で自己相関を考慮
